## Getting started

1. Install dependencies:
`yarn install`

2. Run `docker-compose up -d`

3. Start the watchers: `yarn start`

## How to

- Run db migrations: `yarn db:migration:migrate`
- Create new migration: `yarn db:migration:generate --name create-users-table`
